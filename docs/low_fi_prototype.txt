Low-fi прототип экрана «Каталог игр» (блок-схема)

Текстовая схема:
+---------------------------+
|      [ обложка игры ]     |  <- область изображения
+---------------------------+
| Название / описание       |  <- блок описания
+---------------------------+
| [ Предыдущая ] [ Следующая ] | <- панель управления
+---------------------------+

Рисунок-прототип (ручной эскиз) для отчёта по лабораторной:
  см. файл low_fi_prototype.png в этой папке (docs/).
